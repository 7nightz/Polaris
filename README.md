## Terminal Deployment 
> **Note:**
> We require NodeJS V18 or higher

### Steps
1. Clone the repo
```bash
git clone https://github.com/Skoolgq/Polaris.git
```

2. Enter the directory
```bash
cd Polaris
```

3. Install Dependencies
```bash
npm install
```

4. Run it!
```bash
npm start
```

5. Visit [localhost:8080](http://localhost:8080) for your local Polaris!
